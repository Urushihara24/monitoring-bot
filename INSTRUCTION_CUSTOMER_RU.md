# Простая инструкция (актуальная)

## 1) Что делает бот
- Следит за ценой конкурента по ссылке.
- Автоматически меняет вашу цену по выбранному режиму.
- Работает сразу с несколькими товарами в каждом профиле (`GGSEL` / `DIGISELLER`).
- Может отправлять авто-инструкции покупателям в чат заказа.

## 2) Главное меню
- `📊 Статус` — краткая сводка по текущему активному товару.
- `📦 Товары` — добавить товар и/или привязать ссылку конкурента.
- `⬅ Пред. товар` / `➡ След. товар` — переключение активного товара.
- `🧩 Профиль` — выбор площадки (`GGSEL` или `DIGISELLER`).
- `⚙ Настройки` — режим цены, авто-режим, инструкции.

## 3) Быстрый старт
1. Выберите площадку: `🧩 Профиль`.
2. Добавьте товар: `📦 Товары` -> отправьте `ID`.
3. Привяжите конкурента:
   - сразу: `ID https://ссылка-на-конкурента`
   - либо позже тем же способом.
4. Включите автоцену: `⚙ Настройки` -> `🔔 Авто: ВКЛ`.
5. Проверьте в `📊 Статус`, что видны:
   - ваш товар,
   - цена конкурента,
   - нужный режим.

## 4) Как управлять товарами
- Добавить товар: отправьте `ID` в меню `📦 Товары`.
- Добавить пару `товар + конкурент`: отправьте `ID URL`.
- Показать список: отправьте `list` в меню `📦 Товары`.
- Удалить товар: `⚙ Настройки` -> `🗑 Удалить товар`, затем `ID` или `active`.

Важно:
- Бот мониторит **все товары из списка**.
- Настройки меняются для **активного товара** (который выбран кнопками `⬅/➡`).

## 5) Режимы цены
Переключение: `⚙ Настройки` -> `🔀 Режим`.

- `Следование`
  - цена = точно как у конкурента (4 знака).
  - пример: `0.3560 -> 0.3560`

- `Демпинг`
  - от витринной цены (2 знака): `round(цена_конкурента, 2) - 0.0051`
  - пример: `0.3505 -> 0.35 -> 0.3449`

- `Повышение`
  - от витринной цены (2 знака): `round(цена_конкурента, 2) + 0.0049`
  - пример: `0.3505 -> 0.35 -> 0.3549`

## 6) Авто-инструкции в чат заказа
Включаются в `⚙ Настройки`:

- `💬 Инструкции: ВКЛ/ВЫКЛ`
- `📭 Только пустой чат: ВКЛ/ВЫКЛ`
  - ВКЛ: отправка только если чат пустой.
  - ВЫКЛ: отправка возможна и в непустой чат.
- `🧠 Умный непустой: ВКЛ/ВЫКЛ`
  - дополнительный фильтр для непустых чатов (чтобы отсеивать «завершённые» диалоги).
- `🧭 Режим отправки` (по активному товару):
  - `После заказа` — отправлять сразу после появления заказа.
  - `После 1-го сообщения` — отправлять только когда есть первое сообщение покупателя.
  - `Только при коде` — отправлять только когда в сообщении покупателя найден код.

## 7) Правила инструкций по параметрам
`⚙ Настройки` -> `📝 Правила инстр.`

Как работает:
- Бот показывает варианты параметров товара.
- Включение/выключение делается кнопками под сообщением.
- Если включено хотя бы одно правило — отправка идёт только по включённым вариантам.
- Если для правила не задан свой текст, берётся текст инструкции из товара.

Опциональные команды в этом режиме:
- `text <номер> <текст>` — задать свой текст для конкретного варианта.
- `clear <номер>` — очистить свой текст.
- `reset` — сбросить правила.
- `done` — выйти из редактора.

## 8) Как читать статус
В `📊 Статус` смотрите:
- `Площадка` и `Активный товар`.
- `Моя цена` и `Выставлено ботом`.
- `Цена конкурента` + `URL`.
- `Режим` и `Авто: ВКЛ/ВЫКЛ`.
- Блок авто-инструкций (если поддерживается профилем).

## 9) Если «не сработало»
- Нет изменения цены:
  - проверьте, что есть привязанный `URL` конкурента;
  - проверьте, что включен `🔔 Авто`;
  - проверьте, что вы на нужном профиле и нужном активном товаре.
- Не ушла инструкция:
  - проверьте `💬 Инструкции: ВКЛ`;
  - проверьте `🧭 Режим отправки`;
  - проверьте `📭 Только пустой чат` и правила `📝 Правила инстр.`.
